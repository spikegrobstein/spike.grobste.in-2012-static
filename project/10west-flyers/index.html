10West Flyers

Categories

  • Illustration

  • Web Graphics

This is a collection of flyers I created for a bar/lounge in southern New Jersey.

All files were created in 30 minutes to 2 hours.

Goals for this project were to keep a central theme between designs and have all flyers look like they belong together.

  • Doublesided 4" x 6"
  • UV-coated cardstock.
  • CMYK printing with PMS color matching. (300dpi)
  • Adobe Illustrator for layouts
  • Adobe Photoshop for backgrounds
